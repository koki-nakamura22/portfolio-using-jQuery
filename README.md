# Environmental construction
Please execute following commands.
```
npm install webpack-dev-server -g
npm link webpack
npm i -D webpack-cli
```

# How to execute developer server
Please execute following command.
```
npm run start
```

# How to build
## Development mode
Please execute following command.
```
npm run build:dev
```

## Production mode
Please execute following command.
```
npm run build:pro
```
